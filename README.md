# Rentify - Campus Marketplace 🎓

Rentify is a premium, high-performance peer-to-peer campus marketplace designed for students to buy, rent, and exchange skills.

## Features
- **Modern UI**: Buttery smooth animations with GSAP and CSS 3D transforms.
- **Dark Mode**: Fully supports native dark mode with neon green accents.
- **Firebase Integrated**: Serverless architecture using Firestore for data and Auth for secure logins.
- **Deployment Ready**: Optimized for Vercel with clean URLs.

## Deployment Instructions (Vercel)
1. **Connect Database**: Open `firebase-config.js` and replace the placeholder keys with your actual Firebase project credentials.
2. **Push to GitHub**: Initialize a repository and push this clean codebase.
3. **Deploy**: Import the repository into Vercel. It will automatically detect the configuration and deploy.

## Local Development
- Use a local server (e.g., `npx http-server`) to run the project.
- Ensure you have configured Firebase locally to view live items.

## Built With
- **Logic**: Vanilla JavaScript
- **Database**: Firebase Firestore & Auth
- **Animations**: GSAP & Vanilla-Tilt
- **Styling**: Pure CSS (Custom Theme)
